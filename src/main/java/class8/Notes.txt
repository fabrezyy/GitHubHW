Break keyword:
    Break keyword breaks the loop. We always use break keyword with if conditions once the condition is met it stops
    the loop even if there are still iterations remaining.
Why should we use break:
    If we use break we can save a lot of unnecessary processing by terminating the loop once the goal is complete.
Continue:
    Used to skip unnecessary calculations.
    if an element meets a condition that element is skipped and we move to the next element.
Why should we use continue?
    To save resources to avoid unnecessary calculations.
Note:
    Break and continue keyword are always written with if conditions inside the loop.
